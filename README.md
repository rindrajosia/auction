## How to use


- git clone https://github.com/rindrajosia/scopic.git 
- cd scopic
- git checkout back
- composer install
- npm install
- cp .env.example .env
- create mysql databe called "scopic"
- php artisan migrate
- php artisan db:seed --class=RoleSeeder
- php artisan db:seed --class=UserSeeder
- php artisan db:seed --class=ProductSeeder
- php artisan serve
